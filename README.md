# To compile
1. Fork this repo
2. Use GitHub Action/Workfloｗ
3. Get .ipa or .tipa file
4. Sideload the app (SideStore/TrollStore/LiveContainer/etc.)
# How to use
Open the app on your device. Run `python3 bleakTest.py` then pair your PC and your apple device together. Select your device number in that command prompt. Finish
# Known issue
This repo tested in `iPhone 6s iOS 15.5` + `iPhone 11 iOS 17.6.1` + `iPad Air 3 iOS 16.6.1` but seems any iPhone devices except iPhone 6s has a problem with the UI.
